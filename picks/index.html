Model Picks

A transparent lean for every game across moneyline, total, and run line — team quality, recent form, and the pitching matchup, with the math shown on every card.

How it works: team strength blends Pythagorean win% (from run differential — 75%) with last-10 form (25%). The two strengths are combined head-to-head (log5) and the home team gets the historical ~54% home-field bump. Then the starting pitcher matchup shifts the number: each run of season-ERA advantage moves the favorite by roughly 4–5 points (starters with under 20 innings count as league average). Total and run line use a lighter normal-approximation model on the same team/pitcher inputs. With your Odds API key saved, every market is compared to its no-vig probability — Value only appears when the model beats the price by 3+ points.
