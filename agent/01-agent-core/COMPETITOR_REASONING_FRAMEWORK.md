# Competitor Reasoning Framework

Competitor analysis is allowed. Copying is not.

The purpose is to study reasoning, not steal picks.

## What to collect

For each competitor pick:

- Competitor/source
- Publish time
- Game
- Market
- Pick
- Price quoted
- Reasoning summary
- Main data used
- Reasoning type
- Whether LyDia independently agrees
- Whether line moved after publication
- Closing-line value
- Result
- Lesson learned

## Reasoning tags

Use one or more:

```text
SP_EDGE
OFFENSE_SPLIT
BULLPEN_EDGE
LINEUP_NEWS
WEATHER_TOTAL
MARKET_MOVE
PRICE_VALUE
BUY_LOW
FADE_PUBLIC
INJURY_REACTION
NARRATIVE_ONLY
HEAVY_FAVORITE_CHASE
MODEL_PROJECTION
RECENT_FORM
```

## Competitor reasoning grades

```text
A = data-backed, price-aware, independently useful
B = reasonable but incomplete
C = mostly narrative or missing price discipline
D = weak, copied, hype-based, or unsupported
```

## Key rule

Competitor agreement can increase confidence only after LyDia has its own edge.

Competitor agreement cannot create the edge.
